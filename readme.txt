
This is base code of EControl Syntax Editor SDK (econtrol.ru)
For Lazarus (1.5+)

Code contains: syntax parser (lexer engine), syntax manager (list of lexers), helper lists, regex engine (lexer needs it).
Code does not contain: SyntaxMemo control, helper controls (popup lists) etc, all visual controls.
Code modified! You can use it with ATSynEdit only. 
(memostrings object deleted, replaced with ATStringbuffer object [same methods], less code).
